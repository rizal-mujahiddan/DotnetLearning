| Action          | Command / Shortcut                  | Result                                      |
| --------------- | ----------------------------------- | ------------------------------------------- |
| Start Watcher   | dotnet watch run (Ctrl + Shift + B) | App starts, Polling active, Port 5051 busy. |
| Start Debugging | F5 (Attach)                         | Breakpoints turn solid red.                 |
| Apply Changes   | Ctrl + S (Save)                     | Hot Reload updates code; Cookies stay!      |
| Stop Debugger   | Shift + F5                          | Breakpoints stop working; App stays alive.  |
| Stop Everything | Ctrl + C in Terminal                | Port 5051 is freed.                         |
